ECWM604 Advanced Web Technologies

Week 3 Tutorial

Exercise 1

Install CodeIgniter in your public_html directories if you have not already done so. A youtube video on installing CodeIgniter in your university account is here.

Check that it runs using this form of URL: https://STUDENT_ID.users.ecs.westminster.ac.uk/ecwm604/index.php/Welcome/index (assuming that you have installed CI into a sub-directory of your public_html directory called 'ecwm604').

Exercise 2

For this exercise, you are going to write your own view to replace the default one.

Find the default Welcome controller in the application/controllers directory and open it in a text editor.

Edit the index() method so that instead of loading the view called 'welcome_message', it loads a view called 'myview'. Save the file.

Find the application/views directory. Open an editor and create an HTML page. Save it as 'myview.php' in the views directory. Reload the URL https://STUDENT_ID.users.ecs.westminster.ac.uk/ecwm604/Welcome/index and check you see your HTML page instead of the welcome_message page.

Exercise 3

Create a Model class called Star_model which includes a method called lookup(). The method should return an array with data about a celebrity (perhaps name, url of an image, date of birth etc.)

Create a view called 'star_display'. The HTML code in the view should display the data about a celebrity.

Create a Controller called Stars and add an index() method. In the index() method, use the Star_model Model class and the star_display View to generate a page about a celebrity.

Test your code to make sure it works, i.e., that when you call https://STUDENT_ID.users.ecs.westminster.ac.uk/ecwm604/Stars/index, you see a page about your celebrity.
